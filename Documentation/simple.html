Command Line Tools

The Simple PHP Framework includes command-line tools that can be run from the terminal to interact with the application. Some of the key commands include:

Create Command

To run create, use the php simple create command:

php simple create::controller test  // Create a new Controller "TestController"
php simple create::model test      // Create a new Model "Test"
php simple create::migration test // Create a new Migration "Test"
php simple create::seeder test    // Create a new Seeder "TestSeeder"

Migrate Command

To run migrate, use the php simple migrate command:

php simple migrate::up       // Run the 'up' migration to apply changes to the database
php simple migrate::down     // Run the 'down' migration to revert the last migration
php simple migrate::refresh  // Refresh the database (run down then up migrations)
php simple migrate::status   // Check the status of migrations (whether they have been run)

Seed Command

To run seed, use the php simple seed command:

php simple seed::run    // Run the database seeder to populate the database with test data
php simple seed::reset  // Reset the seeders and reseed the database
php simple seed::refresh // Refresh the seeders, effectively re-running the reset and seeding process
php simple seed::status  // Check the status of the seeders
